# Hola Mundo en PHP

## Descripción
Este proyecto consiste en una aplicación simple desarrollada en **PHP** que muestra en pantalla el mensaje:

Hola mundo, soy Edwar Chamalé

Además, la página incluye una imagen generada con inteligencia artificial.

PHP es un **lenguaje interpretado**, por lo que **no es necesario compilar el código para ejecutarlo**.

## Prerrequisitos

Para ejecutar este proyecto se necesita:

- PHP 7 o superior
- Navegador web
- Terminal

Para verificar si PHP está instalado:

php -v
